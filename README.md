# E-commerce Database Design

## Description
This project is a relational database design for an E-commerce system. It includes multiple tables with proper relationships using primary and foreign keys.

## Tables
- Users
- Products
- Orders
- Order_Items
- Payments

## Tools Used
- MySQL
- dbdiagram.io

## Features
- Normalized database structure
- One-to-many and many-to-many relationships
- ER Diagram included
